## Ground Rules

1. **Alternate class or base class — pick one.** Don't mix features between a LaserLlama alternate class and its base-class counterpart.
2. **Link your source.** On your PC page, note exactly which document and version you're using. Saves us both an argument at level 7.
3. **Rulings beat rules.** If something's ambiguous mid-session, I'll make a fast call and we'll settle it properly after.
4. **Balance flags are welcome.** If something plays broken or feels dead on arrival, tell me. Adjusting it is fair game and not a personal failing.
